# Android Liquid Glass (Backdrop)

![frontPhoto](artworks/banner.jpg)

A customizable Liquid Glass effect library for Jetpack Compose.

## Docs

[![](https://jitpack.io/v/onedeba7/backdrop.svg)](https://jitpack.io/#onedeba7/backdrop) [![Android CI](https://github.com/onedeba7/AndroidLiquidGlass/actions/workflows/android.yml/badge.svg)](https://github.com/onedeba7/AndroidLiquidGlass/actions/workflows/android.yml)

[Documentation](https://kyant.gitbook.io/backdrop)

## Components

The library does not include any high-level components; you will need to create your own.
Below are some example components:

- [LiquidButton](/app/src/main/java/com/github/debacodex/backdrop/catalog/components/LiquidButton.kt)
- [LiquidToggle](/app/src/main/java/com/github/debacodex/catalog/components/LiquidToggle.kt)
- [LiquidSlider](/app/src/main/java/com/github/debacodex/catalog/components/LiquidSlider.kt)
- [LiquidBottomTabs](/app/src/main/java/com/com/github/debacodex/catalog/components/LiquidBottomTabs.kt)
